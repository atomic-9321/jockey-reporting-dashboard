# Positive Language Rules for Jockey Reporting

## Core Principle
All reporting language must be constructive, actionable, and optimistic. We frame results as learnings and opportunities, never as failures or losses.

## Word Substitutions
- "failure" -> "learning"
- "failed" -> "tested"
- "waste" / "wasted" -> "investment" / "invested"
- "lose" / "loss" -> "redirect" / "opportunity"
- "decline" / "declined" -> "adjustment" / "adjusted"
- "drop" / "dropped" -> "shift" / "shifted"
- "decrease" -> "optimization opportunity"
- "underperforming" -> "opportunity area"
- "poor" / "bad" -> "developing" / "evolving"
- "problem" / "issue" -> "opportunity" / "focus area"
- "down" -> "adjusting"

## Framing Rules
1. Always lead with what IS working before addressing what needs attention
2. Frame declines as "optimization opportunities" not problems
3. Reference specific data points — never vague claims
4. Always end with actionable next steps
5. Use "investment" instead of "spend" or "cost"
6. Use "testing" and "learning" when referencing experiments
7. Highlight "growth potential" instead of "shortcomings"

## Trend Language
- Positive trend: "Strong momentum", "Accelerating", "Building on previous gains"
- Flat trend: "Stable foundation", "Consistent performance", "Maintaining baseline"
- Negative trend: "Optimization opportunity", "Room for growth", "Area for focused attention"

## Example Reframes
- BAD: "ROAS dropped 30% this week, campaigns are failing"
- GOOD: "ROAS shifted 30% this week, presenting an optimization opportunity. Based on the data, reallocating investment toward the top-performing creatives could accelerate recovery."

- BAD: "Ad spend was wasted on underperforming campaigns"
- GOOD: "Investment in testing campaigns provided valuable learnings. The data suggests focusing future investment on the 3 top-performing creatives."
